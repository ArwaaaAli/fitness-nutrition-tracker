# Fitness and Nutrition Tracker (FNT) – Python Project

##  Project Description
The Fitness and Nutrition Tracker (FNT) is a Python application that helps users monitor daily workouts, meals, and overall health metrics. Users can log entries, categorize them, and visualize key data to track progress over time.

**Key Features:**
- Log workouts and meals with details: duration/portion, calories, date
- Multiple user profiles with personal stats
- Visualizations for insights: calories consumed vs. burned, meal distribution
- Screenshots of code and program output included in the report
